# Ingenieur
L'objectif de ce repo est de parser le document data/annuaire88.pdf qui contient une mini biographie des élèves de l'Ecole Centrale entre 1832 et 1888. On souhaite extraire pour chaque ingénieur les informations suivantes :
* NOM
* DOMAINE DE PROFESSION
* ANNEE DE PROMOTION
* RECOMPENSES
* LIEU(X) DE TRAVAIL
